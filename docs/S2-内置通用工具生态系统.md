# S2：内置通用工具生态系统 — 补充里程碑深度开发计划

> **里程碑**：S2 · **优先级**：P0（与 M2 并行）  
> **前置依赖**：M1（事件流管道架构）、M2（声明式能力注册表）  
> **预计代码量**：~4500 行新增 + ~200 行重构  
> **涉及包**：`@orbit/agent-core`（主战场）  
> **定位**：填补 M1-M7 里程碑中「工具只有注册框架，无任何实现」的空白

---

## 动机

Orbit 当前 `agent-core` 只有空壳的 `ToolRegistry`（165 行）。用户可以注册自定义工具，但框架本身不提供任何内置工具。这导致一个关键问题：**离开了 Orbit 宿主应用的特定 Capability，Agent 什么都做不了。**

参考 Hermes-agent 的设计：原生内置 50+ 通用工具（web 搜索、文件操作、终端命令、浏览器自动化等），全部与业务无关，可独立使用。这使得 Hermes 的 Agent 在纯对话场景下就能完成搜索、编码、文件管理等强大任务。

**Orbit 需要同样的能力层。**

本里程碑与 M1-M7 正交：M2 定义了 `Capability<Input, Output>` 的声明式框架，S2 则在这个框架上实现 16 个内置通用工具。M3 定义了 Safety Gate 责任链，S2 的工具安全模块直接集成到该责任链中。

### 与现有里程碑的关系

```
M1 (事件流)
  ↓

  ↓                        ↑
M3 (Safety Gate) ──────────┘ (安全集成)
  ↓
M4 (记忆系统) ←── S2.memory_tool (记忆工具)
  ↓
M6 (多Agent编排) ←── S2.delegate_task (任务委派)
```

## 第一章：工具架构设计

### 1.1 设计目标

1. **零依赖独立运行**：每个内置工具只依赖 Node.js 标准库或可选的轻量级 peer dependency，不依赖 Orbit 宿主应用
2. **兼容 M2 Capability 接口**：每个工具同时实现 `BuiltinTool` 接口和 M2 的 `Capability<Input, Output>` 泛型接口
3. **Toolset 分组**：工具按功能域分组，支持按场景组合启用/禁用
4. **生命周期管理**：工具有明确的注册 → 可用性检查 → 执行 → 结果处理生命周期
5. **并发安全声明**：每个工具声明自己的并发安全级别，由调度器统一管理

### 1.2 BuiltinTool 基础接口

`BuiltinTool` 是 S2 工具生态的核心抽象。它既是独立的工具定义，也能通过 `adaptToolToCapability()` 适配为 M2 的 `Capability<Input, Output>`。

```typescript
// packages/agent-core/src/tools/types.ts

import { z, type ZodSchema } from 'zod';
import type {
  AgentDomain,
  RiskLevel,
  ApprovalPolicy,
  ExecutionMode,
  ScopeLimit,
  DataBoundary,
} from '../types.js';

// ---- 并发安全等级（复用 M2 定义）----
export const CONCURRENCY_LEVELS = ['never_parallel', 'path_scoped', 'parallel_safe'] as const;
export type ConcurrencyLevel = (typeof CONCURRENCY_LEVELS)[number];

// ---- 工具集标识 ----
export const TOOLSET_IDS = [
  'web',
  'file',
  'terminal',
  'browser',
  'code_execution',
  'delegation',
  'vision',
  'interaction',
  'planning',
  'memory',
] as const;
export type ToolsetId = (typeof TOOLSET_IDS)[number];

// ---- 工具元数据 ----
export interface ToolMetadata {
  /** 工具的唯一标识符 */
  readonly name: string;
  /** 所属工具集 */
  readonly toolset: ToolsetId;
  /** 人类可读描述（用于 LLM function calling） */
  readonly description: string;
  /** 显示用 emoji */
  readonly emoji: string;
  /** 工具所属的 Orbit Agent 域 */
  readonly domain: AgentDomain;
  /** 静态风险等级（可被 riskLevel(input) 覆盖） */
  readonly defaultRiskLevel: RiskLevel;
  /** 静态审批策略（可被动态覆盖） */
  readonly defaultApprovalPolicy: ApprovalPolicy;
  /** 执行模式 */
  readonly executionMode: ExecutionMode;
  /** 作用域限制 */
  readonly scopeLimit: ScopeLimit;
  /** 数据边界 */
  readonly dataBoundary: DataBoundary;
  /** 并发安全等级 */
  readonly concurrency: ConcurrencyLevel;
  /** 结果最大字符数（超出则截断） */
  readonly maxResultSizeChars: number;
  /** 需要的环境变量列表（用于 check_fn） */
  readonly requiresEnv: readonly string[];
  /** 需要的可执行文件列表（用于 check_fn） */
  readonly requiresBinaries: readonly string[];
  /** 是否为异步工具 */
  readonly isAsync: boolean;
}

// ---- 工具执行上下文 ----
export interface ToolExecutionContext {
  /** 当前工作目录 */
  readonly cwd: string;
  /** 当前 session ID */
  readonly sessionId: string;
  /** 当前 run ID */
  readonly runId: string;
  /** 触发此工具的 Agent 域 */
  readonly agentDomain: AgentDomain;
  /** AbortSignal，用于取消传播 */
  readonly signal: AbortSignal;
  /** 超时（毫秒） */
  readonly timeoutMs: number;
  /** 环境变量（已过滤的安全子集） */
  readonly env: Readonly<Record<string, string>>;
  /** 允许的路径前缀列表（文件操作用） */
  readonly allowedPaths: readonly string[];
  /** 是否为沙箱模式 */
  readonly sandboxed: boolean;
}

// ---- 工具执行结果 ----
export interface ToolExecutionResult<Output = unknown> {
  readonly success: boolean;
  readonly output: Output;
  readonly rawText: string;
  readonly truncated: boolean;
  readonly durationMs: number;
  readonly error?: string;
  readonly metadata?: Readonly<Record<string, unknown>>;
}

// ---- BuiltinTool 核心接口 ----
export interface BuiltinTool<Input = unknown, Output = unknown> {
  /** 工具元数据 */
  readonly metadata: ToolMetadata;

  /** Zod 输入验证 schema */
  readonly inputSchema: ZodSchema<Input>;

  /** JSON Schema（用于 OpenAI function calling） */
  readonly jsonSchema: Readonly<Record<string, unknown>>;

  /**
   * 可用性检查。检查当前运行环境是否满足工具要求。
   * 例如：web_search 需要网络访问和 API key，browser 需要 Playwright。
   */
  checkAvailability(ctx: ToolExecutionContext): Promise<boolean>;

  /**
   * 根据具体输入动态评估风险等级。
   * 例如：terminal 执行 `ls` 是 R0-read，执行 `rm -rf` 是 R3-external-write。
   */
  riskLevel(input: Input): RiskLevel;

  /**
   * 根据具体输入动态评估并发安全性。
   * 例如：file_read 是 parallel_safe，file_write 对同一路径是 path_scoped。
   */
  concurrencySafety(input: Input): ConcurrencyLevel;

  /**
   * 对于 path_scoped 工具，返回输入所涉及的路径。
   * 并发调度器用此信息判断是否存在路径冲突。
   */
  scopePaths(input: Input): readonly string[];

  /**
   * 执行工具逻辑。
   * @param input 经 Zod 验证后的输入
   * @param ctx 执行上下文
   */
  execute(input: Input, ctx: ToolExecutionContext): Promise<ToolExecutionResult<Output>>;
}
```

### 1.3 工具到 M2 Capability 的适配层

```typescript
// packages/agent-core/src/tools/tool-to-capability-adapter.ts

import type { BuiltinTool, ToolExecutionContext } from './types.js';
import type {
  Capability, ExecutionContext, CapabilityEvent, CostProfile,
} from '../capability-types.js';
import type { RiskLevel, ApprovalPolicy } from '../types.js';

/**
 * 将 BuiltinTool 适配为 M2 Capability 接口。
 * 使得内置工具可以无缝参与 CapabilityRegistry 和 StreamingCapabilityExecutor。
 */
export function adaptToolToCapability<Input, Output>(
  tool: BuiltinTool<Input, Output>,
): Capability<Input, Output> {
  const meta = tool.metadata;

  return {
    id: `builtin.${meta.toolset}.${meta.name}`,
    domain: meta.domain,
    description: meta.description,
    inputSchema: tool.inputSchema,
    outputSchema: undefined,

    staticRiskLevel: meta.defaultRiskLevel,
    staticApprovalPolicy: meta.defaultApprovalPolicy,
    executionMode: meta.executionMode,
    scopeLimit: meta.scopeLimit,
    dataBoundary: meta.dataBoundary,

    riskLevel(input: Input): RiskLevel {
      return tool.riskLevel(input);
    },

    approvalPolicy(_input: Input): ApprovalPolicy {
      return tool.metadata.defaultApprovalPolicy;
    },

    concurrencySafety(input: Input): 'safe' | 'path_scoped' | 'exclusive' {
      const level = tool.concurrencySafety(input);
      const mapping = {
        parallel_safe: 'safe' as const,
        path_scoped: 'path_scoped' as const,
        never_parallel: 'exclusive' as const,
      };
      return mapping[level];
    },

    scopePaths(input: Input): readonly string[] {
      return tool.scopePaths(input);
    },

    costProfile(): CostProfile {
      return {
        estimatedTokens: { min: 0, max: 0 },
        estimatedDurationMs: { min: 100, max: 30000 },
        estimatedCostCents: 0,
        externalBilling: meta.requiresEnv.length > 0,
      };
    },

    async *execute(
      input: Input,
      ctx: ExecutionContext,
    ): AsyncGenerator<CapabilityEvent<Output>> {
      yield {
        kind: 'started',
        capabilityId: `builtin.${meta.toolset}.${meta.name}`,
        timestamp: new Date().toISOString(),
        message: `Executing ${meta.emoji} ${meta.name}...`,
      };

      const toolCtx: ToolExecutionContext = {
        cwd: process.cwd(),
        sessionId: ctx.sessionId,
        runId: ctx.runId,
        agentDomain: ctx.agentDomain,
        signal: ctx.signal,
        timeoutMs: ctx.budget.remainingTimeMs,
        env: process.env as Record<string, string>,
        allowedPaths: [process.cwd()],
        sandboxed: false,
      };

      try {
        const result = await tool.execute(input, toolCtx);
        if (result.success) {
          yield {
            kind: 'completed',
            capabilityId: `builtin.${meta.toolset}.${meta.name}`,
            timestamp: new Date().toISOString(),
            partialResult: result.output as Partial<Output>,
          };
        } else {
          yield {
            kind: 'failed',
            capabilityId: `builtin.${meta.toolset}.${meta.name}`,
            timestamp: new Date().toISOString(),
            error: result.error ?? 'Unknown error',
          };
        }
      } catch (err: unknown) {
        yield {
          kind: 'failed',
          capabilityId: `builtin.${meta.toolset}.${meta.name}`,
          timestamp: new Date().toISOString(),
          error: err instanceof Error ? err.message : String(err),
        };
      }
    },
  };
}
```

### 1.4 AbstractBuiltinTool 基类

为减少每个工具的样板代码，提供一个抽象基类：

```typescript
// packages/agent-core/src/tools/abstract-builtin-tool.ts

import type { ZodSchema } from 'zod';
import type {
  BuiltinTool,
  ToolMetadata,
  ToolExecutionContext,
  ToolExecutionResult,
  ConcurrencyLevel,
} from './types.js';
import type { RiskLevel } from '../types.js';

/**
 * 内置工具的抽象基类。
 * 提供默认的 riskLevel()、concurrencySafety()、scopePaths()、checkAvailability() 实现。
 * 子类只需实现 execute() 和提供 metadata/schema。
 */
export abstract class AbstractBuiltinTool<Input = unknown, Output = unknown>
  implements BuiltinTool<Input, Output>
{
  abstract readonly metadata: ToolMetadata;
  abstract readonly inputSchema: ZodSchema<Input>;
  abstract readonly jsonSchema: Readonly<Record<string, unknown>>;

  async checkAvailability(ctx: ToolExecutionContext): Promise<boolean> {
    for (const envVar of this.metadata.requiresEnv) {
      if (!ctx.env[envVar]) return false;
    }
    return true;
  }

  riskLevel(_input: Input): RiskLevel {
    return this.metadata.defaultRiskLevel;
  }

  concurrencySafety(_input: Input): ConcurrencyLevel {
    return this.metadata.concurrency;
  }

  scopePaths(_input: Input): readonly string[] {
    return [];
  }

  abstract execute(
    input: Input,
    ctx: ToolExecutionContext,
  ): Promise<ToolExecutionResult<Output>>;

  protected ok(output: Output, rawText: string, durationMs: number): ToolExecutionResult<Output> {
    return { success: true, output, rawText, truncated: false, durationMs };
  }

  protected fail(error: string, durationMs: number): ToolExecutionResult<Output> {
    return {
      success: false,
      output: undefined as unknown as Output,
      rawText: '',
      truncated: false,
      durationMs,
      error,
    };
  }

  protected truncateResult(text: string): { text: string; truncated: boolean } {
    const max = this.metadata.maxResultSizeChars;
    if (text.length <= max) return { text, truncated: false };
    const suffix = `\n\n... [truncated: showing ${max} of ${text.length} chars]`;
    return { text: text.slice(0, max) + suffix, truncated: true };
  }
}
```

### 1.5 工具生命周期

```
┌─────────────────────────────────────────────────────────────┐
│                    Tool Lifecycle                             │
│                                                              │
│  register()    ┌────────────┐   checkAvailability()          │
│  ───────────►  │ Registered  │ ──────────────────►           │
│                └────────────┘                                │
│                      │                                       │
│               ┌──────┴──────┐                                │
│               ▼             ▼                                │
│        ┌───────────┐  ┌──────────────┐                       │
│        │   Ready    │  │ Unavailable  │                       │
│        │ (in list)  │  │ (hidden)     │                       │
│        └─────┬─────┘  └──────────────┘                       │
│              │ execute()                                     │
│              ▼                                               │
│        ┌───────────┐    ┌─────────────────┐                  │
│        │ Executing  │──►│ Result Processing│                  │
│        └───────────┘    │ (truncate/store) │                  │
│                         └─────────────────┘                  │
└─────────────────────────────────────────────────────────────┘
```

**阶段说明：**

1. **注册（Register）**：工具实例通过 `BuiltinToolManager.register()` 注册到 ToolRegistry。同时自动调用 `adaptToolToCapability()` 注册到 CapabilityRegistry（如果 M2 已实现）。
2. **可用性检查（Availability Check）**：每次 `getDefinitions()` 或工具调用前，执行 `checkAvailability()`。不可用的工具不出现在 LLM 的 tool 列表中。
3. **就绪（Ready）**：工具已通过检查，可被 LLM 调用。
4. **执行（Execute）**：LLM 发起 tool_call → 输入 Zod 验证 → SafetyGate 检查 → execute()。
5. **结果处理（Result Processing）**：结果截断 → 存储（可选）→ 转换为 ToolResult 或 CapabilityEvent 返回。

### 1.6 BuiltinToolManager — 工具管理器

```typescript
// packages/agent-core/src/tools/builtin-tool-manager.ts

import type { BuiltinTool, ToolExecutionContext, ToolsetId } from './types.js';
import type { ToolRegistry } from '../tool-registry.js';
import type { ToolsetConfig } from './toolset-config.js';

/**
 * 管理所有内置工具的生命周期。
 * - 批量注册工具到 ToolRegistry
 * - 根据 ToolsetConfig 启用/禁用工具集
 * - 在 dispatch 时执行输入验证、安全检查、结果截断
 */
export class BuiltinToolManager {
  private readonly tools = new Map<string, BuiltinTool>();
  private readonly registry: ToolRegistry;
  private readonly enabledToolsets = new Set<ToolsetId>();

  constructor(registry: ToolRegistry) {
    this.registry = registry;
  }

  register(tool: BuiltinTool): void {
    const name = tool.metadata.name;
    if (this.tools.has(name)) {
      throw new Error(`Built-in tool "${name}" is already registered`);
    }
    this.tools.set(name, tool);

    this.registry.register(
      {
        name: tool.metadata.name,
        domain: tool.metadata.domain,
        description: tool.metadata.description,
        inputSchema: tool.jsonSchema,
        riskLevel: tool.metadata.defaultRiskLevel,
        approvalPolicy: tool.metadata.defaultApprovalPolicy,
        executionMode: tool.metadata.executionMode,
        scopeLimit: tool.metadata.scopeLimit,
        dataBoundary: tool.metadata.dataBoundary,
      },
      async (args) => this.dispatch(name, args),
      () => this.enabledToolsets.has(tool.metadata.toolset),
    );
  }

  registerAll(tools: readonly BuiltinTool[]): void {
    for (const tool of tools) this.register(tool);
  }

  applyConfig(config: ToolsetConfig): void {
    this.enabledToolsets.clear();
    for (const id of config.enabledToolsets) this.enabledToolsets.add(id);
  }

  getToolsByToolset(toolsetId: ToolsetId): readonly BuiltinTool[] {
    return [...this.tools.values()].filter((t) => t.metadata.toolset === toolsetId);
  }

  getTool(name: string): BuiltinTool | undefined {
    return this.tools.get(name);
  }

  isEnabled(name: string): boolean {
    const tool = this.tools.get(name);
    return tool ? this.enabledToolsets.has(tool.metadata.toolset) : false;
  }

  private async dispatch(name: string, args: Record<string, unknown>): Promise<string> {
    const tool = this.tools.get(name);
    if (!tool) throw new Error(`Built-in tool "${name}" not found`);

    // 1. Zod 输入验证
    const parseResult = tool.inputSchema.safeParse(args);
    if (!parseResult.success) {
      const errors = parseResult.error.issues
        .map((i: { path: (string | number)[]; message: string }) =>
          `${i.path.join('.')}: ${i.message}`)
        .join('; ');
      return JSON.stringify({ success: false, error: `Invalid input: ${errors}` });
    }

    // 2. 构建执行上下文
    const ctx: ToolExecutionContext = {
      cwd: process.cwd(),
      sessionId: '',
      runId: '',
      agentDomain: tool.metadata.domain,
      signal: AbortSignal.timeout(60_000),
      timeoutMs: 60_000,
      env: process.env as Record<string, string>,
      allowedPaths: [process.cwd()],
      sandboxed: false,
    };

    // 3. 可用性检查
    const available = await tool.checkAvailability(ctx);
    if (!available) {
      return JSON.stringify({
        success: false,
        error: `Tool "${name}" is unavailable. Required env: [${tool.metadata.requiresEnv.join(', ')}]`,
      });
    }

    // 4. 执行
    const result = await tool.execute(parseResult.data, ctx);

    // 5. 结果截断
    let outputText = result.rawText;
    let truncated = result.truncated;
    if (outputText.length > tool.metadata.maxResultSizeChars) {
      const max = tool.metadata.maxResultSizeChars;
      outputText = outputText.slice(0, max) + `\n\n... [truncated: ${max}/${outputText.length} chars]`;
      truncated = true;
    }

    return JSON.stringify({
      success: result.success,
      output: outputText,
      truncated,
      durationMs: result.durationMs,
      error: result.error,
    });
  }
}
```

### 2.1 web_search — 网页搜索

**定位**：多 provider 网页搜索，支持 Google、Bing、DuckDuckGo、Tavily。Agent 获取实时信息的基础能力。

```typescript
// packages/agent-core/src/tools/web/web-search.ts

import { z } from 'zod';
import { AbstractBuiltinTool } from '../abstract-builtin-tool.js';
import type { ToolMetadata, ToolExecutionContext, ToolExecutionResult } from '../types.js';

// ---- Input / Output 类型 ----

const WebSearchInputSchema = z.object({
  query: z.string().min(1).max(500).describe('搜索查询字符串'),
  provider: z
    .enum(['google', 'bing', 'duckduckgo', 'tavily'])
    .default('duckduckgo')
    .describe('搜索引擎 provider'),
  maxResults: z.number().int().min(1).max(20).default(5).describe('返回结果数量上限'),
  language: z.string().default('en').describe('搜索语言代码（ISO 639-1）'),
  safeSearch: z.boolean().default(true).describe('是否开启安全搜索'),
});

type WebSearchInput = z.infer<typeof WebSearchInputSchema>;

interface WebSearchResult {
  readonly title: string;
  readonly url: string;
  readonly snippet: string;
  readonly publishedDate?: string;
}

interface WebSearchOutput {
  readonly results: readonly WebSearchResult[];
  readonly query: string;
  readonly provider: string;
  readonly totalResults?: number;
}

// ---- JSON Schema（OpenAI function calling 格式）----

const WEB_SEARCH_JSON_SCHEMA = {
  type: 'object',
  properties: {
    query: { type: 'string', description: '搜索查询字符串', minLength: 1, maxLength: 500 },
    provider: {
      type: 'string',
      enum: ['google', 'bing', 'duckduckgo', 'tavily'],
      default: 'duckduckgo',
      description: '搜索引擎 provider',
    },
    maxResults: {
      type: 'integer', minimum: 1, maximum: 20, default: 5,
      description: '返回结果数量上限',
    },
    language: { type: 'string', default: 'en', description: '搜索语言代码' },
    safeSearch: { type: 'boolean', default: true, description: '是否开启安全搜索' },
  },
  required: ['query'],
} as const;

// ---- 工具实现 ----

export class WebSearchTool extends AbstractBuiltinTool<WebSearchInput, WebSearchOutput> {
  readonly metadata: ToolMetadata = {
    name: 'web_search',
    toolset: 'web',
    description:

  async execute(

      const results = await this.searchByProvider(input, ctx);
      const output: WebSearchOutput = {
        results,
        query: input.query,
        provider: input.provider,
        totalResults: results.length,
      };

      const rawText = results
        .map((r, i) => `${i + 1}. [${r.title}](${r.url})\n   ${r.snippet}`)

      return this.ok(output, rawText, Date.now() - start);

      return this.fail(
        `Web search failed: ${err instanceof Error ? err.message : String(err)}`,
        Date.now() - start,
      );
    }

  private async searchByProvider(
    input: WebSearchInput,
    ctx: ToolExecutionContext,
  ): Promise<WebSearchResult[]> {
    switch (input.provider) {
      case 'duckduckgo':
        return this.searchDuckDuckGo(input, ctx);
      case 'tavily':
        return this.searchTavily(input, ctx);
      case 'google':
        return this.searchGoogle(input, ctx);
      case 'bing':
        return this.searchBing(input, ctx);
      default:
        return this.searchDuckDuckGo(input, ctx);
    }

  private async searchDuckDuckGo(
    input: WebSearchInput,
    ctx: ToolExecutionContext,
  ): Promise<WebSearchResult[]> {
    // DuckDuckGo Instant Answer API（免费，无需 key）
    const params = new URLSearchParams({
      q: input.query,
      format: 'json',
      no_html: '1',
      skip_disambig: '1',
    });

    const response = await fetch(
      `https://api.duckduckgo.com/?${params.toString()}`,
      { signal: ctx.signal },
    );
    const data = await response.json() as Record<string, unknown>;

    // 解析 DuckDuckGo 的 RelatedTopics 为统一格式
    const topics = (data.RelatedTopics as Array<Record<string, unknown>>) ?? [];
    return topics
      .filter((t) => t.FirstURL && t.Text)
      .slice(0, input.maxResults)
      .map((t) => ({
        title: String(t.Text).slice(0, 100),
        url: String(t.FirstURL),
        snippet: String(t.Text),
      }));
  }

  private async searchTavily(
    input: WebSearchInput,
    ctx: ToolExecutionContext,
  ): Promise<WebSearchResult[]> {
    const apiKey = ctx.env['TAVILY_API_KEY'];
    if (!apiKey) throw new Error('TAVILY_API_KEY not set');

    const response = await fetch('https://api.tavily.com/search', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({
        api_key: apiKey,
        query: input.query,
        max_results: input.maxResults,
        search_depth: 'basic',
      }),
      signal: ctx.signal,
    });
    const data = await response.json() as { results: Array<{ title: string; url: string; content: string }> };

    return data.results.map((r) => ({
      title: r.title,
      url: r.url,
      snippet: r.content,
    }));
  }

  private async searchGoogle(
    input: WebSearchInput,
    ctx: ToolExecutionContext,
  ): Promise<WebSearchResult[]> {
    const apiKey = ctx.env['GOOGLE_API_KEY'];
    const cseId = ctx.env['GOOGLE_CSE_ID'];
    if (!apiKey || !cseId) throw new Error('GOOGLE_API_KEY and GOOGLE_CSE_ID required');

    const params = new URLSearchParams({
      key: apiKey, cx: cseId, q: input.query,
      num: String(input.maxResults), hl: input.language,
      safe: input.safeSearch ? 'active' : 'off',
    });

    const response = await fetch(
      `https://www.googleapis.com/customsearch/v1?${params.toString()}`,
      { signal: ctx.signal },
    );
    const data = await response.json() as { items?: Array<{ title: string; link: string; snippet: string }> };

    return (data.items ?? []).map((item) => ({
      title: item.title,
      url: item.link,
      snippet: item.snippet,
    }));
  }

  private async searchBing(
    input: WebSearchInput,
    ctx: ToolExecutionContext,
  ): Promise<WebSearchResult[]> {
    const apiKey = ctx.env['BING_API_KEY'];
    if (!apiKey) throw new Error('BING_API_KEY not set');

    const params = new URLSearchParams({
      q: input.query, count: String(input.maxResults),
      mkt: input.language === 'en' ? 'en-US' : input.language,
      safeSearch: input.safeSearch ? 'Moderate' : 'Off',
    });

    const response = await fetch(
      `https://api.bing.microsoft.com/v7.0/search?${params.toString()}`,
      { headers: { 'Ocp-Apim-Subscription-Key': apiKey }, signal: ctx.signal },
    );
    const data = await response.json() as {
      webPages?: { value: Array<{ name: string; url: string; snippet: string; dateLastCrawled?: string }> };
    };

    return (data.webPages?.value ?? []).map((r) => ({
      title: r.name,
      url: r.url,
      snippet: r.snippet,
      publishedDate: r.dateLastCrawled,
    }));
  }
```

### 2.2 web_extract — 网页内容提取

**定位**：给定 URL，提取页面主体内容并转换为 Markdown。使用 Readability 算法过滤导航、广告等噪音。

```typescript
// packages/agent-core/src/tools/web/web-extract.ts

import { z } from 'zod';
import { AbstractBuiltinTool } from '../abstract-builtin-tool.js';
import type { ToolMetadata, ToolExecutionContext, ToolExecutionResult } from '../types.js';

const WebExtractInputSchema = z.object({
  url: z.string().url().describe('要提取内容的 URL'),
  format: z.enum(['markdown', 'text', 'html']).default('markdown').describe('输出格式'),
  maxLength: z.number().int().min(100).max(50000).default(10000).describe('最大提取字符数'),
  includeImages: z.boolean().default(false).describe('是否包含图片链接'),
  includeLinks: z.boolean().default(true).describe('是否保留超链接'),
});

interface WebExtractOutput {
  readonly title: string;
  readonly content: string;
  readonly url: string;
  readonly wordCount: number;
  readonly extractedAt: string;
}

const WEB_EXTRACT_JSON_SCHEMA = {
  type: 'object',
  properties: {
    url: { type: 'string', format: 'uri', description: '要提取内容的 URL' },
    format: { type: 'string', enum: ['markdown', 'text', 'html'], default: 'markdown' },
    maxLength: { type: 'integer', minimum: 100, maximum: 50000, default: 10000 },
    includeImages: { type: 'boolean', default: false },
    includeLinks: { type: 'boolean', default: true },
  },
  required: ['url'],
} as const;

export class WebExtractTool extends AbstractBuiltinTool<WebExtractInput, WebExtractOutput> {
  readonly metadata: ToolMetadata = {
    name: 'web_extract',
    toolset: 'web',
    description:
      'Extract the main content from a web page URL and convert it to clean Markdown. ' +
      'Strips navigation, ads, and other noise using Readability algorithm.',
    emoji: '📄',
    domain: 'research',
    defaultRiskLevel: 'R2-external-read',
    defaultApprovalPolicy: 'A1-transparent',
    executionMode: 'async',
    scopeLimit: 'global',
    dataBoundary: 'can-egress',
    concurrency: 'parallel_safe',
    maxResultSizeChars: 15000,
    requiresEnv: [],
    requiresBinaries: [],
    isAsync: true,
  };

  readonly inputSchema = WebExtractInputSchema;
  readonly jsonSchema = WEB_EXTRACT_JSON_SCHEMA;

  async execute(
    input: WebExtractInput,
    ctx: ToolExecutionContext,
  ): Promise<ToolExecutionResult<WebExtractOutput>> {
    const start = Date.now();

    try {
      // 1. Fetch HTML
      const response = await fetch(input.url, {
        headers: {
          'User-Agent': 'OrbitBot/1.0 (compatible; research assistant)',
          'Accept': 'text/html,application/xhtml+xml',
        },
        signal: ctx.signal,
      });

      if (!response.ok) {
        return this.fail(`HTTP ${response.status}: ${response.statusText}`, Date.now() - start);
      }

      const html = await response.text();

      // 2. 简易 Readability 提取（生产环境应使用 @mozilla/readability）
      const content = this.extractContent(html, input);
      const title = this.extractTitle(html);

      const output: WebExtractOutput = {
        title,
        content: content.slice(0, input.maxLength),
        url: input.url,
        wordCount: content.split(/\s+/).length,
        extractedAt: new Date().toISOString(),
      };

      return this.ok(output, `# ${title}\n\n${output.content}`, Date.now() - start);
    } catch (err: unknown) {
      return this.fail(
        `Extraction failed: ${err instanceof Error ? err.message : String(err)}`,
        Date.now() - start,
      );
    }
  }

  private extractTitle(html: string): string {
    const match = html.match(/<title[^>]*>([\s\S]*?)<\/title>/i);
    return match ? match[1].trim().replace(/\s+/g, ' ') : 'Untitled';
  }

  private extractContent(html: string, input: WebExtractInput): string {
    // 移除 script、style、nav、header、footer 标签
    let cleaned = html
      .replace(/<script[\s\S]*?<\/script>/gi, '')
      .replace(/<style[\s\S]*?<\/style>/gi, '')
      .replace(/<nav[\s\S]*?<\/nav>/gi, '')
      .replace(/<header[\s\S]*?<\/header>/gi, '')
      .replace(/<footer[\s\S]*?<\/footer>/gi, '')
      .replace(/<aside[\s\S]*?<\/aside>/gi, '');

    if (input.format === 'text') {
      // 纯文本：移除所有 HTML 标签
      return cleaned.replace(/<[^>]+>/g, ' ').replace(/\s+/g, ' ').trim();
    }

    // Markdown 转换
    // 标题转换
    cleaned = cleaned.replace(/<h([1-6])[^>]*>([\s\S]*?)<\/h\1>/gi, (_m, level, text) => {
      const hashes = '#'.repeat(parseInt(level));
      return `\n${hashes} ${text.replace(/<[^>]+>/g, '').trim()}\n`;
    });
    // 段落
    cleaned = cleaned.replace(/<p[^>]*>([\s\S]*?)<\/p>/gi, '\n$1\n');
    // 链接
    if (input.includeLinks) {
      cleaned = cleaned.replace(/<a[^>]+href="([^"]*)"[^>]*>([\s\S]*?)<\/a>/gi, '[$2]($1)');
    }
    // 粗体、斜体
    cleaned = cleaned.replace(/<(strong|b)[^>]*>([\s\S]*?)<\/\1>/gi, '**$2**');
    cleaned = cleaned.replace(/<(em|i)[^>]*>([\s\S]*?)<\/\1>/gi, '*$2*');
    // 列表
    cleaned = cleaned.replace(/<li[^>]*>([\s\S]*?)<\/li>/gi, '- $1');
    // 代码
    cleaned = cleaned.replace(/<code[^>]*>([\s\S]*?)<\/code>/gi, '`$1`');
    // 移除剩余标签
    cleaned = cleaned.replace(/<[^>]+>/g, '');
    // 清理空白
    cleaned = cleaned.replace(/\n{3,}/g, '\n\n').trim();

    return cleaned;
  }
}
```

### 2.3 terminal — Shell 命令执行

**定位**：在用户工作目录下执行 Shell 命令。支持子进程管理、超时控制、输出截断。这是 Agent 操作本地环境的核心能力。

```typescript
// packages/agent-core/src/tools/terminal/terminal.ts

import { z } from 'zod';
import { spawn } from 'node:child_process';
import path from 'node:path';
import { AbstractBuiltinTool } from '../abstract-builtin-tool.js';
import type {
  ToolMetadata, ToolExecutionContext, ToolExecutionResult, ConcurrencyLevel,
} from '../types.js';
import type { RiskLevel } from '../../types.js';
import { DESTRUCTIVE_PATTERNS, classifyCommandRisk } from '../safety/command-classifier.js';

const TerminalInputSchema = z.object({
  command: z.string().min(1).max(5000).describe('要执行的 Shell 命令'),
  cwd: z.string().optional().describe('工作目录（默认为当前项目目录）'),
  timeoutMs: z.number().int().min(1000).max(300000).default(30000).describe('超时时间（毫秒）'),
  shell: z.enum(['bash', 'sh', 'zsh', 'powershell']).default('bash').describe('Shell 类型'),
});

type TerminalInput = z.infer<typeof TerminalInputSchema>;

interface TerminalOutput {
  readonly exitCode: number;
  readonly stdout: string;
  readonly stderr: string;
  readonly timedOut: boolean;
  readonly command: string;
}

const TERMINAL_JSON_SCHEMA = {
  type: 'object',
  properties: {
    command: { type: 'string', description: '要执行的 Shell 命令', minLength: 1, maxLength: 5000 },
    cwd: { type: 'string', description: '工作目录（默认为当前项目目录）' },
    timeoutMs: {
      type: 'integer', minimum: 1000, maximum: 300000, default: 30000,
      description: '超时时间（毫秒）',
    },
    shell: {
      type: 'string', enum: ['bash', 'sh', 'zsh', 'powershell'], default: 'bash',
    },
  },
  required: ['command'],
} as const;

export class TerminalTool extends AbstractBuiltinTool<TerminalInput, TerminalOutput> {
  readonly metadata: ToolMetadata = {
    name: 'terminal',
    toolset: 'terminal',
    description:
      'Execute a shell command in the current working directory. ' +
      'Returns stdout, stderr, and exit code. Supports timeout and process management.',
    emoji: '💻',
    domain: 'ops',
    defaultRiskLevel: 'R1-internal-write',
    defaultApprovalPolicy: 'A1-transparent',
    executionMode: 'async',
    scopeLimit: 'current-project',
    dataBoundary: 'local-only',
    concurrency: 'never_parallel',
    maxResultSizeChars: 20000,
    requiresEnv: [],
    requiresBinaries: [],
    isAsync: true,
  };

  readonly inputSchema = TerminalInputSchema;
  readonly jsonSchema = TERMINAL_JSON_SCHEMA;

  /**
   * 根据命令内容动态评估风险等级。
   * 只读命令（ls, cat, echo）→ R0-read
   * 写入命令（mkdir, touch）→ R1-internal-write
   * 破坏性命令（rm -rf, git reset --hard）→ R3-external-write
   */
  riskLevel(input: TerminalInput): RiskLevel {
    return classifyCommandRisk(input.command);
  }

  concurrencySafety(_input: TerminalInput): ConcurrencyLevel {
    // Shell 命令默认不可并行（可能互相影响文件系统状态）
    return 'never_parallel';
  }

  async execute(
    input: TerminalInput,
    ctx: ToolExecutionContext,
  ): Promise<ToolExecutionResult<TerminalOutput>> {
    const start = Date.now();

    // 安全检查：工作目录必须在允许范围内
    const workDir = input.cwd
      ? path.resolve(ctx.cwd, input.cwd)
      : ctx.cwd;

    if (!ctx.allowedPaths.some((p) => workDir.startsWith(p))) {
      return this.fail(
        `Working directory "${workDir}" is outside allowed paths: [${ctx.allowedPaths.join(', ')}]`,
        Date.now() - start,
      );
    }

    const timeout = Math.min(input.timeoutMs, ctx.timeoutMs);

    return new Promise((resolve) => {
      let stdout = '';
      let stderr = '';
      let timedOut = false;

      const shellBin = input.shell === 'powershell' ? 'pwsh' : input.shell;
      const child = spawn(shellBin, ['-c', input.command], {
        cwd: workDir,
        timeout,
        env: { ...ctx.env, PATH: process.env['PATH'] ?? '' },
        stdio: ['ignore', 'pipe', 'pipe'],
      });

      child.stdout.on('data', (chunk: Buffer) => {
        stdout += chunk.toString();
        // 超大输出实时截断
        if (stdout.length > this.metadata.maxResultSizeChars * 2) {
          child.kill('SIGTERM');
        }
      });

      child.stderr.on('data', (chunk: Buffer) => {
        stderr += chunk.toString();
      });

      child.on('error', (err) => {
        resolve(this.fail(`Process error: ${err.message}`, Date.now() - start));
      });

      child.on('close', (code) => {
        const output: TerminalOutput = {
          exitCode: code ?? 1,
          stdout: stdout.slice(0, this.metadata.maxResultSizeChars),
          stderr: stderr.slice(0, 5000),
          timedOut,
          command: input.command,
        };

        const rawText = [
          `$ ${input.command}`,
          stdout ? `\n${stdout}` : '',
          stderr ? `\n[stderr]\n${stderr}` : '',
          `\n[exit code: ${code ?? 1}]`,
        ].join('');

        resolve(this.ok(output, rawText, Date.now() - start));
      });

      // AbortSignal 联动
      if (ctx.signal) {
        ctx.signal.addEventListener('abort', () => {
          timedOut = true;
          child.kill('SIGTERM');
        });
      }
    });
  }
}
```

### 2.4 file_read — 文件读取

**定位**：安全地读取文件内容，支持行号范围、编码检测、二进制文件识别。

```typescript

import { z } from 'zod';
import fs from 'node:fs/promises';
import path from 'node:path';
import { AbstractBuiltinTool } from '../abstract-builtin-tool.js';

    path: { type: 'string', description: '文件路径' },

    encoding: { type: 'string', enum: ['utf-8', 'ascii', 'latin1', 'base64'], default: 'utf-8' },
  },
  required: ['path'],
} as const;

export class FileReadTool extends AbstractBuiltinTool<FileReadInput, FileReadOutput> {
  readonly metadata: ToolMetadata = {
    name: 'file_read',
    toolset: 'file',
    description:
      'Read the contents of a file. Supports line range selection, encoding detection, ' +
      'and binary file identification. Paths are validated for security.',
    emoji: '📖',
    domain: 'reading',
    defaultRiskLevel: 'R0-read',
    defaultApprovalPolicy: 'A0-auto',
    executionMode: 'sync',
    scopeLimit: 'current-project',
    dataBoundary: 'local-only',
    concurrency: 'parallel_safe',
    maxResultSizeChars: 30000,
    requiresEnv: [],
    requiresBinaries: [],
    isAsync: false,
  };

  readonly inputSchema = FileReadInputSchema;
  readonly jsonSchema = FILE_READ_JSON_SCHEMA;

  scopePaths(input: FileReadInput): readonly string[] {
    return [input.path];
  }

  async execute(
    input: FileReadInput,
    ctx: ToolExecutionContext,
  ): Promise<ToolExecutionResult<FileReadOutput>> {
    const start = Date.now();

    // 1. 路径安全验证
    const resolvedPath = path.resolve(ctx.cwd, input.path);
    const pathCheck = validatePath(resolvedPath, ctx.allowedPaths);
    if (!pathCheck.valid) {
      return this.fail(`Path rejected: ${pathCheck.reason}`, Date.now() - start);
    }

    try {
      // 2. 检查文件存在性
      const stat = await fs.stat(resolvedPath);
      if (!stat.isFile()) {
        return this.fail(`"${input.path}" is not a file`, Date.now() - start);
      }

      // 3. 二进制文件检测
      if (stat.size > 0) {
        const header = Buffer.alloc(512);
        const fd = await fs.open(resolvedPath, 'r');
        await fd.read(header, 0, 512, 0);
        await fd.close();
        if (this.isBinary(header)) {
          return this.fail(
            `"${input.path}" appears to be a binary file (${stat.size} bytes)`,
            Date.now() - start,
          );
        }
      }

      // 4. 读取内容
      const raw = await fs.readFile(resolvedPath, input.encoding as BufferEncoding);
      const lines = raw.split('\n');
      const totalLines = lines.length;

      const startLine = input.startLine ?? 1;
      const endLine = input.endLine ?? totalLines;
      const selectedLines = lines.slice(startLine - 1, endLine);

      // 5. 添加行号
      const content = selectedLines
        .map((line, i) => `${startLine + i}. ${line}`)
        .join('\n');

      const { text: truncatedContent, truncated } = this.truncateResult(content);

      const output: FileReadOutput = {
        content: truncatedContent,
        path: resolvedPath,
        totalLines,
        startLine,
        endLine: Math.min(endLine, totalLines),
        sizeBytes: stat.size,
      };

      return {
        success: true,
        output,
        rawText: truncatedContent,
        truncated,
        durationMs: Date.now() - start,
      };
    } catch (err: unknown) {
      return this.fail(
        `Read failed: ${err instanceof Error ? err.message : String(err)}`,
        Date.now() - start,
      );
    }
  }

  private isBinary(buffer: Buffer): boolean {
    // 检查前 512 字节中是否有 NULL 字节（简单启发式）
    for (let i = 0; i < Math.min(buffer.length, 512); i++) {
      if (buffer[i] === 0) return true;
    }
    return false;
  }
}
```

### 2.5 file_write — 文件写入

```typescript
    encoding: { type: 'string', enum: ['utf-8', 'ascii', 'latin1'], default: 'utf-8' },
  },
  required: ['path', 'content'],
} as const;

export class FileWriteTool extends AbstractBuiltinTool<FileWriteInput, FileWriteOutput> {
  readonly metadata: ToolMetadata = {
    name: 'file_write',
    toolset: 'file',
    description:
      'Write content to a file with atomic operation. Creates backup before overwriting. ' +
      'Supports dry-run mode for diff preview.',
    emoji: '✏️',
    domain: 'writing',
    defaultRiskLevel: 'R1-internal-write',
    defaultApprovalPolicy: 'A1-transparent',
    executionMode: 'sync',
    scopeLimit: 'current-project',
    dataBoundary: 'local-only',
    concurrency: 'path_scoped',
    maxResultSizeChars: 5000,
    requiresEnv: [],
    requiresBinaries: [],
    isAsync: false,
  };

  readonly inputSchema = FileWriteInputSchema;
  readonly jsonSchema = FILE_WRITE_JSON_SCHEMA;

  concurrencySafety(_input: FileWriteInput): ConcurrencyLevel {
    return 'path_scoped';
  }

  scopePaths(input: FileWriteInput): readonly string[] {
    return [input.path];
  }

  async execute(
    input: FileWriteInput,
    ctx: ToolExecutionContext,
  ): Promise<ToolExecutionResult<FileWriteOutput>> {
    const start = Date.now();

    const resolvedPath = path.resolve(ctx.cwd, input.path);
    const pathCheck = validatePath(resolvedPath, ctx.allowedPaths);
    if (!pathCheck.valid) {
      return this.fail(`Path rejected: ${pathCheck.reason}`, Date.now() - start);
    }

    try {
      // 检查文件是否已存在
      let existingContent: string | null = null;
      let created = true;
      try {
        existingContent = await fs.readFile(resolvedPath, input.encoding as BufferEncoding);
        created = false;
      } catch {
        // 文件不存在，将创建新文件
      }

      // 生成 diff
      const diff = existingContent !== null
        ? this.generateSimpleDiff(existingContent, input.content, input.path)
        : `+++ new file: ${input.path}\n+ ${input.content.split('\n').join('\n+ ')}`;

      // Dry-run 模式
      if (input.dryRun) {
        const output: FileWriteOutput = {
          path: resolvedPath,
          bytesWritten: 0,
          created: false,
          diff,
        };
        return this.ok(output, `[dry-run] Diff preview:\n${diff}`, Date.now() - start);
      }

      // 创建父目录
      if (input.createDirectories) {
        await fs.mkdir(path.dirname(resolvedPath), { recursive: true });
      }

      // 创建备份
      let backupPath: string | undefined;
      if (input.backup && existingContent !== null) {
        backupPath = `${resolvedPath}.bak`;
        await fs.writeFile(backupPath, existingContent, input.encoding as BufferEncoding);
      }

      // 原子写入（先写 .tmp 再 rename）
      const tmpPath = `${resolvedPath}.orbit-tmp`;
      await fs.writeFile(tmpPath, input.content, input.encoding as BufferEncoding);
      await fs.rename(tmpPath, resolvedPath);

      const output: FileWriteOutput = {
        path: resolvedPath,
        bytesWritten: Buffer.byteLength(input.content, input.encoding as BufferEncoding),
        backupPath,
        created,
        diff,
      };

      const rawText = created
        ? `Created ${resolvedPath} (${output.bytesWritten} bytes)`
        : `Updated ${resolvedPath} (${output.bytesWritten} bytes)${backupPath ? `, backup: ${backupPath}` : ''}`;

      return this.ok(output, rawText, Date.now() - start);
    } catch (err: unknown) {
      return this.fail(
        `Write failed: ${err instanceof Error ? err.message : String(err)}`,
        Date.now() - start,
      );
    }
  }

  private generateSimpleDiff(oldContent: string, newContent: string, filePath: string): string {
    const oldLines = oldContent.split('\n');
    const newLines = newContent.split('\n');
    const diffLines: string[] = [`--- a/${filePath}`, `+++ b/${filePath}`];

    const maxLines = Math.max(oldLines.length, newLines.length);
    for (let i = 0; i < maxLines; i++) {
      const oldLine = i < oldLines.length ? oldLines[i] : undefined;
      const newLine = i < newLines.length ? newLines[i] : undefined;

      if (oldLine === newLine) {
        diffLines.push(` ${oldLine ?? ''}`);
      } else {
        if (oldLine !== undefined) diffLines.push(`-${oldLine}`);
        if (newLine !== undefined) diffLines.push(`+${newLine}`);
      }
    }

    return diffLines.join('\n');
  }
}
```

### 2.6 file_search — 文件搜索

**定位**：封装 ripgrep/grep 进行文件内容搜索，同时支持 glob 文件名匹配。

```typescript
// packages/agent-core/src/tools/file/file-search.ts

import { z } from 'zod';
import { spawn } from 'node:child_process';
import path from 'node:path';
import { AbstractBuiltinTool } from '../abstract-builtin-tool.js';
import type { ToolMetadata, ToolExecutionContext, ToolExecutionResult } from '../types.js';

const FileSearchInputSchema = z.object({
  pattern: z.string().min(1).describe('搜索模式（正则表达式或字面量字符串）'),
  path: z.string().default('.').describe('搜索的根目录'),
  glob: z.string().optional().describe('文件名 glob 过滤（例如 "*.ts"）'),
  caseSensitive: z.boolean().default(false).describe('是否区分大小写'),
  maxResults: z.number().int().min(1).max(100).default(20).describe('最大结果数'),
  contextLines: z.number().int().min(0).max(5).default(2).describe('匹配行前后的上下文行数'),
  mode: z.enum(['content', 'files', 'count']).default('content').describe('搜索模式'),
});

type FileSearchInput = z.infer<typeof FileSearchInputSchema>;

interface FileSearchMatch {
  readonly file: string;
  readonly line: number;
  readonly content: string;
  readonly contextBefore: readonly string[];
  readonly contextAfter: readonly string[];
}

interface FileSearchOutput {
  readonly matches: readonly FileSearchMatch[];
  readonly totalMatches: number;
  readonly filesSearched: number;
}

const FILE_SEARCH_JSON_SCHEMA = {
  type: 'object',
  properties: {
    pattern: { type: 'string', description: '搜索模式（正则表达式）' },
    path: { type: 'string', default: '.', description: '搜索根目录' },
    glob: { type: 'string', description: '文件名 glob 过滤' },
    caseSensitive: { type: 'boolean', default: false },
    maxResults: { type: 'integer', minimum: 1, maximum: 100, default: 20 },
    contextLines: { type: 'integer', minimum: 0, maximum: 5, default: 2 },
    mode: { type: 'string', enum: ['content', 'files', 'count'], default: 'content' },
  },
  required: ['pattern'],
} as const;

export class FileSearchTool extends AbstractBuiltinTool<FileSearchInput, FileSearchOutput> {
  readonly metadata: ToolMetadata = {
    name: 'file_search',
    toolset: 'file',
    description:
      'Search for patterns in files using ripgrep. Supports regex patterns, ' +
      'glob filtering, context lines, and multiple output modes.',
    emoji: '🔎',
    domain: 'reading',
    defaultRiskLevel: 'R0-read',
    defaultApprovalPolicy: 'A0-auto',
    executionMode: 'async',
    scopeLimit: 'current-project',
    dataBoundary: 'local-only',
    concurrency: 'parallel_safe',
    maxResultSizeChars: 15000,
    requiresEnv: [],
    requiresBinaries: [],
    isAsync: true,
  };

  readonly inputSchema = FileSearchInputSchema;
  readonly jsonSchema = FILE_SEARCH_JSON_SCHEMA;

  async execute(
    input: FileSearchInput,
    ctx: ToolExecutionContext,
  ): Promise<ToolExecutionResult<FileSearchOutput>> {
    const start = Date.now();
    const searchPath = path.resolve(ctx.cwd, input.path);

    try {
      const args = this.buildRipgrepArgs(input);
      const output = await this.runRipgrep(args, searchPath, ctx);
      const matches = this.parseRipgrepOutput(output, input.maxResults);

      const result: FileSearchOutput = {
        matches,
        totalMatches: matches.length,
        filesSearched: new Set(matches.map((m) => m.file)).size,
      };

      const rawText = matches
        .map((m) => `${m.file}:${m.line}: ${m.content}`)
        .join('\n');

      return this.ok(result, rawText || 'No matches found.', Date.now() - start);
    } catch (err: unknown) {
      // ripgrep 退出码 1 = 无匹配，不是错误
      const msg = err instanceof Error ? err.message : String(err);
      if (msg.includes('exit code 1')) {
        const emptyResult: FileSearchOutput = { matches: [], totalMatches: 0, filesSearched: 0 };
        return this.ok(emptyResult, 'No matches found.', Date.now() - start);
      }
      return this.fail(`Search failed: ${msg}`, Date.now() - start);
    }
  }

  private buildRipgrepArgs(input: FileSearchInput): string[] {
    const args = ['--json', '--max-count', String(input.maxResults)];

    if (!input.caseSensitive) args.push('-i');
    if (input.contextLines > 0) {
      args.push('-C', String(input.contextLines));
    }
    if (input.glob) {
      args.push('-g', input.glob);
    }
    if (input.mode === 'files') {
      args.push('-l');
    } else if (input.mode === 'count') {
      args.push('-c');
    }

    args.push('--', input.pattern);
    return args;
  }

  private runRipgrep(
    args: string[],
    cwd: string,
    ctx: ToolExecutionContext,
  ): Promise<string> {
    return new Promise((resolve, reject) => {
      // 尝试 rg (ripgrep)，回退到 grep
      const child = spawn('rg', args, {
        cwd,
        timeout: ctx.timeoutMs,
        stdio: ['ignore', 'pipe', 'pipe'],
      });

      let stdout = '';
      let stderr = '';

      child.stdout.on('data', (chunk: Buffer) => { stdout += chunk.toString(); });
      child.stderr.on('data', (chunk: Buffer) => { stderr += chunk.toString(); });

      child.on('error', () => {
        // ripgrep 不可用，回退到 grep
        this.runGrep(args, cwd, ctx).then(resolve).catch(reject);
      });

      child.on('close', (code) => {
        if (code === 0 || code === 1) resolve(stdout);
        else reject(new Error(`rg exited with code ${code}: ${stderr}`));
      });
    });
  }

  private runGrep(
    _args: string[],
    cwd: string,
    ctx: ToolExecutionContext,
  ): Promise<string> {
    // grep 的简化回退实现
    return new Promise((resolve, reject) => {
      const child = spawn('grep', ['-rn', '--include=*', '.'], {
        cwd,
        timeout: ctx.timeoutMs,
        stdio: ['ignore', 'pipe', 'pipe'],
      });

      let stdout = '';
      child.stdout.on('data', (chunk: Buffer) => { stdout += chunk.toString(); });
      child.on('close', (code) => {
        if (code === 0 || code === 1) resolve(stdout);
        else reject(new Error(`grep exited with code ${code}`));
      });
    });
  }

  private parseRipgrepOutput(output: string, maxResults: number): FileSearchMatch[] {
    const matches: FileSearchMatch[] = [];
    const lines = output.split('\n').filter(Boolean);

    for (const line of lines) {
      if (matches.length >= maxResults) break;
      try {
        const parsed = JSON.parse(line) as {
          type: string;
          data: {
            path?: { text: string };
            line_number?: number;
            lines?: { text: string };
          };
        };
        if (parsed.type === 'match' && parsed.data.path) {
          matches.push({
            file: parsed.data.path.text,
            line: parsed.data.line_number ?? 0,
            content: parsed.data.lines?.text?.trim() ?? '',
            contextBefore: [],
            contextAfter: [],
          });
        }
      } catch {
        // 非 JSON 行（grep 回退格式）: file:line:content
        const match = line.match(/^(.+?):(\d+):(.+)/);
        if (match) {
          matches.push({
            file: match[1],
            line: parseInt(match[2], 10),
            content: match[3].trim(),
            contextBefore: [],
            contextAfter: [],
          });
        }
      }
    }

    return matches;
  }
}
```

### 2.7 file_patch — 文件补丁

**定位**：基于 fuzzy 匹配的文件修改，比 file_write 更精确——只修改指定片段，无需重写整个文件。

```typescript
// packages/agent-core/src/tools/file/file-patch.ts

import { z } from 'zod';
import fs from 'node:fs/promises';
import path from 'node:path';
import { AbstractBuiltinTool } from '../abstract-builtin-tool.js';
import type {
  ToolMetadata, ToolExecutionContext, ToolExecutionResult, ConcurrencyLevel,
} from '../types.js';
import { validatePath } from '../safety/path-validator.js';

const FilePatchInputSchema = z.object({
  path: z.string().min(1).describe('要修改的文件路径'),
  oldContent: z.string().min(1).describe('要被替换的原始内容（支持 fuzzy 匹配）'),
  newContent: z.string().describe('替换后的新内容'),
  backup: z.boolean().default(true).describe('是否创建备份'),
  fuzzyThreshold: z.number().min(0).max(1).default(0.8).describe('Fuzzy 匹配阈值（0-1）'),
});

type FilePatchInput = z.infer<typeof FilePatchInputSchema>;

interface FilePatchOutput {
  readonly path: string;
  readonly matched: boolean;
  readonly matchConfidence: number;
  readonly lineRange: { start: number; end: number };
  readonly diff: string;
}

const FILE_PATCH_JSON_SCHEMA = {
  type: 'object',
  properties: {
    path: { type: 'string', description: '文件路径' },
    oldContent: { type: 'string', description: '要被替换的原始内容' },
    newContent: { type: 'string', description: '替换后的新内容' },
    backup: { type: 'boolean', default: true },
    fuzzyThreshold: { type: 'number', minimum: 0, maximum: 1, default: 0.8 },
  },
  required: ['path', 'oldContent', 'newContent'],
} as const;

export class FilePatchTool extends AbstractBuiltinTool<FilePatchInput, FilePatchOutput> {
  readonly metadata: ToolMetadata = {
    name: 'file_patch',
    toolset: 'file',
    description:
      'Apply a targeted patch to a file by replacing a specific content block. ' +
      'Uses fuzzy matching to find the target even if whitespace or formatting differs slightly.',
    emoji: '🩹',
    domain: 'writing',
    defaultRiskLevel: 'R1-internal-write',
    defaultApprovalPolicy: 'A1-transparent',
    executionMode: 'sync',
    scopeLimit: 'current-project',
    dataBoundary: 'local-only',
    concurrency: 'path_scoped',
    maxResultSizeChars: 5000,
    requiresEnv: [],
    requiresBinaries: [],
    isAsync: false,
  };

  readonly inputSchema = FilePatchInputSchema;
  readonly jsonSchema = FILE_PATCH_JSON_SCHEMA;

  concurrencySafety(_input: FilePatchInput): ConcurrencyLevel {
    return 'path_scoped';
  }

  scopePaths(input: FilePatchInput): readonly string[] {
    return [input.path];
  }

  async execute(
    input: FilePatchInput,
    ctx: ToolExecutionContext,
  ): Promise<ToolExecutionResult<FilePatchOutput>> {
    const start = Date.now();

    const resolvedPath = path.resolve(ctx.cwd, input.path);
    const pathCheck = validatePath(resolvedPath, ctx.allowedPaths);
    if (!pathCheck.valid) {
      return this.fail(`Path rejected: ${pathCheck.reason}`, Date.now() - start);
    }

    try {
      const content = await fs.readFile(resolvedPath, 'utf-8');

      // 1. 精确匹配
      let matchIndex = content.indexOf(input.oldContent);
      let confidence = 1.0;

      // 2. 如果精确匹配失败，尝试 fuzzy 匹配
      if (matchIndex === -1) {
        const fuzzyResult = this.fuzzyFind(content, input.oldContent, input.fuzzyThreshold);
        if (fuzzyResult) {
          matchIndex = fuzzyResult.index;
          confidence = fuzzyResult.confidence;
        }
      }

      if (matchIndex === -1) {
        return this.fail(
          'Could not find matching content in file. Try adjusting oldContent or lowering fuzzyThreshold.',
          Date.now() - start,
        );
      }

      // 3. 计算行号范围
      const beforeMatch = content.slice(0, matchIndex);
      const startLine = beforeMatch.split('\n').length;
      const matchedText = confidence === 1.0
        ? input.oldContent
        : content.slice(matchIndex, matchIndex + input.oldContent.length);
      const endLine = startLine + matchedText.split('\n').length - 1;

      // 4. 备份
      if (input.backup) {
        await fs.writeFile(`${resolvedPath}.bak`, content, 'utf-8');
      }

      // 5. 应用补丁
      const newContent =
        content.slice(0, matchIndex) +
        input.newContent +
        content.slice(matchIndex + matchedText.length);
      await fs.writeFile(resolvedPath, newContent, 'utf-8');

      const diff = [
        `--- a/${input.path}`,
        `+++ b/${input.path}`,
        `@@ -${startLine},${endLine - startLine + 1} @@`,
        ...matchedText.split('\n').map((l: string) => `-${l}`),
        ...input.newContent.split('\n').map((l: string) => `+${l}`),
      ].join('\n');

      const output: FilePatchOutput = {
        path: resolvedPath,
        matched: true,
        matchConfidence: confidence,
        lineRange: { start: startLine, end: endLine },
        diff,
      };

      return this.ok(output, diff, Date.now() - start);
    } catch (err: unknown) {
      return this.fail(
        `Patch failed: ${err instanceof Error ? err.message : String(err)}`,
        Date.now() - start,
      );
    }
  }

  /**
   * 简单的 fuzzy 查找：将搜索字符串和目标归一化后比较。
   * 生产环境可使用 Levenshtein 距离或更高级的算法。
   */
  private fuzzyFind(
    haystack: string,
    needle: string,
    threshold: number,
  ): { index: number; confidence: number } | null {
    // 归一化：压缩空白、统一换行
    const normalizeWs = (s: string) => s.replace(/\s+/g, ' ').trim();
    const normalizedNeedle = normalizeWs(needle);

    // 滑动窗口比较
    const lines = haystack.split('\n');
    const needleLines = needle.split('\n').length;

    let bestMatch: { index: number; confidence: number } | null = null;

    for (let i = 0; i <= lines.length - needleLines; i++) {
      const candidate = lines.slice(i, i + needleLines).join('\n');
      const normalizedCandidate = normalizeWs(candidate);

      const similarity = this.similarity(normalizedNeedle, normalizedCandidate);
      if (similarity >= threshold && (!bestMatch || similarity > bestMatch.confidence)) {
        // 计算原始字符串中的字节偏移
        const index = haystack.indexOf(candidate);
        if (index !== -1) {
          bestMatch = { index, confidence: similarity };
        }
      }
    }

    return bestMatch;
  }

  private similarity(a: string, b: string): number {
    if (a === b) return 1.0;
    const maxLen = Math.max(a.length, b.length);
    if (maxLen === 0) return 1.0;

    let matches = 0;
    const minLen = Math.min(a.length, b.length);
    for (let i = 0; i < minLen; i++) {
      if (a[i] === b[i]) matches++;
    }
    return matches / maxLen;
  }
}
```

### 2.11 code_execute — 代码执行

**定位**：在沙箱中执行 Python/JavaScript/Shell 代码片段。用于数据处理、计算、验证等场景。

```typescript
// packages/agent-core/src/tools/code/code-execute.ts

import { z } from 'zod';
import { spawn } from 'node:child_process';
import fs from 'node:fs/promises';
import path from 'node:path';
import os from 'node:os';
import { AbstractBuiltinTool } from '../abstract-builtin-tool.js';
import type { ToolMetadata, ToolExecutionContext, ToolExecutionResult } from '../types.js';

const CodeExecuteInputSchema = z.object({
  code: z.string().min(1).max(50000).describe('要执行的代码'),
  language: z.enum(['python', 'javascript', 'typescript', 'shell']).describe('编程语言'),
  timeoutMs: z.number().int().min(1000).max(120000).default(30000).describe('超时（毫秒）'),
  args: z.array(z.string()).default([]).describe('命令行参数'),
});

type CodeExecuteInput = z.infer<typeof CodeExecuteInputSchema>;

interface CodeExecuteOutput {
  readonly stdout: string;
  readonly stderr: string;
  readonly exitCode: number;
  readonly language: string;
  readonly timedOut: boolean;
}

const CODE_EXECUTE_JSON_SCHEMA = {
  type: 'object',
  properties: {
    code: { type: 'string', description: '要执行的代码', maxLength: 50000 },
    language: { type: 'string', enum: ['python', 'javascript', 'typescript', 'shell'] },
    timeoutMs: { type: 'integer', minimum: 1000, maximum: 120000, default: 30000 },
    args: { type: 'array', items: { type: 'string' }, default: [] },
  },
  required: ['code', 'language'],
} as const;

export class CodeExecuteTool extends AbstractBuiltinTool<CodeExecuteInput, CodeExecuteOutput> {
  readonly metadata: ToolMetadata = {
    name: 'code_execute',
    toolset: 'code_execution',
    description:
      'Execute code snippets in Python, JavaScript, TypeScript, or Shell. ' +
      'Runs in a sandboxed environment with timeout limits.',
    emoji: '▶️',
    domain: 'ops',
    defaultRiskLevel: 'R1-internal-write',
    defaultApprovalPolicy: 'A2-confirm',
    executionMode: 'async',
    scopeLimit: 'current-project',
    dataBoundary: 'local-only',
    concurrency: 'never_parallel',
    maxResultSizeChars: 20000,
    requiresEnv: [],
    requiresBinaries: [],
    isAsync: true,
  };

  readonly inputSchema = CodeExecuteInputSchema;
  readonly jsonSchema = CODE_EXECUTE_JSON_SCHEMA;

  async execute(
    input: CodeExecuteInput,
    ctx: ToolExecutionContext,
  ): Promise<ToolExecutionResult<CodeExecuteOutput>> {
    const start = Date.now();

    try {
      const { cmd, args, tempFile } = await this.prepareExecution(input, ctx);

      try {
        const result = await this.runProcess(cmd, args, input.timeoutMs, ctx);

        const output: CodeExecuteOutput = {
          stdout: result.stdout.slice(0, this.metadata.maxResultSizeChars),
          stderr: result.stderr.slice(0, 5000),
          exitCode: result.exitCode,
          language: input.language,
          timedOut: result.timedOut,
        };

        const rawText = [
          `[${input.language}] exit code: ${result.exitCode}`,
          result.stdout ? `\n${result.stdout}` : '',
          result.stderr ? `\n[stderr]\n${result.stderr}` : '',
        ].join('');

        return this.ok(output, rawText, Date.now() - start);
      } finally {
        // 清理临时文件
        if (tempFile) {
          await fs.unlink(tempFile).catch(() => {});
        }
      }
    } catch (err: unknown) {
      return this.fail(
        `Code execution failed: ${err instanceof Error ? err.message : String(err)}`,
        Date.now() - start,
      );
    }
  }

  private async prepareExecution(
    input: CodeExecuteInput,
    ctx: ToolExecutionContext,
  ): Promise<{ cmd: string; args: string[]; tempFile?: string }> {
    const ext = { python: '.py', javascript: '.mjs', typescript: '.ts', shell: '.sh' };
    const tempFile = path.join(ctx.cwd, `.orbit-exec-${Date.now()}${ext[input.language]}`);
    await fs.writeFile(tempFile, input.code, 'utf-8');

    switch (input.language) {
      case 'python':
        return { cmd: 'python3', args: [tempFile, ...input.args], tempFile };
      case 'javascript':
        return { cmd: 'node', args: [tempFile, ...input.args], tempFile };
      case 'typescript':
        return { cmd: 'npx', args: ['tsx', tempFile, ...input.args], tempFile };
      case 'shell':
        return { cmd: 'bash', args: [tempFile, ...input.args], tempFile };
    }
  }

  private runProcess(
    cmd: string,
    args: string[],
    timeoutMs: number,
    ctx: ToolExecutionContext,
  ): Promise<{ stdout: string; stderr: string; exitCode: number; timedOut: boolean }> {
    return new Promise((resolve) => {
      let stdout = '';
      let stderr = '';
      let timedOut = false;

      const child = spawn(cmd, args, {
        cwd: ctx.cwd,
        timeout: timeoutMs,
        stdio: ['ignore', 'pipe', 'pipe'],
        env: { ...ctx.env, PATH: process.env['PATH'] ?? '' },
      });

      child.stdout.on('data', (chunk: Buffer) => { stdout += chunk.toString(); });
      child.stderr.on('data', (chunk: Buffer) => { stderr += chunk.toString(); });

      child.on('close', (code) => {
        resolve({ stdout, stderr, exitCode: code ?? 1, timedOut });
      });

      child.on('error', (err) => {
        resolve({ stdout, stderr: err.message, exitCode: 1, timedOut: false });
      });

      if (ctx.signal) {
        ctx.signal.addEventListener('abort', () => {
          timedOut = true;
          child.kill('SIGTERM');
        });
      }
    });
  }
}
```

### 2.15 todo — 任务规划

**定位**：Agent 的结构化任务管理工具。可读写 todo list，跟踪多步骤任务的进度。

```typescript
// packages/agent-core/src/tools/planning/todo.ts

import { z } from 'zod';
import { AbstractBuiltinTool } from '../abstract-builtin-tool.js';
import type { ToolMetadata, ToolExecutionContext, ToolExecutionResult } from '../types.js';

const TodoInputSchema = z.object({
  action: z.enum(['list', 'add', 'update', 'remove', 'clear']).describe('操作类型'),
  id: z.string().optional().describe('任务 ID（update/remove 时必需）'),
  title: z.string().optional().describe('任务标题（add 时必需）'),
  description: z.string().optional().describe('任务详细描述'),
  status: z.enum(['pending', 'in_progress', 'done', 'blocked']).optional().describe('任务状态'),
  priority: z.enum(['low', 'medium', 'high', 'critical']).optional().describe('优先级'),
});

type TodoInput = z.infer<typeof TodoInputSchema>;

interface TodoItem {
  readonly id: string;
  readonly title: string;
  readonly description?: string;
  readonly status: 'pending' | 'in_progress' | 'done' | 'blocked';
  readonly priority: 'low' | 'medium' | 'high' | 'critical';
  readonly createdAt: string;
}

interface TodoOutput {
  readonly items: readonly TodoItem[];
  readonly action: string;
  readonly affected?: string;
}

const TODO_JSON_SCHEMA = {
  type: 'object',
  properties: {
    action: { type: 'string', enum: ['list', 'add', 'update', 'remove', 'clear'] },
    id: { type: 'string', description: '任务 ID' },
    title: { type: 'string', description: '任务标题' },
    description: { type: 'string' },
    status: { type: 'string', enum: ['pending', 'in_progress', 'done', 'blocked'] },
    priority: { type: 'string', enum: ['low', 'medium', 'high', 'critical'] },
  },
  required: ['action'],
} as const;

export class TodoTool extends AbstractBuiltinTool<TodoInput, TodoOutput> {
  private items = new Map<string, TodoItem>();
  private counter = 0;

  readonly metadata: ToolMetadata = {
    name: 'todo',
    toolset: 'planning',
    description:
      'Manage a structured todo list for tracking multi-step tasks. ' +
      'Supports adding, updating, removing, and listing tasks with priorities.',
    emoji: '📋',
    domain: 'planning',
    defaultRiskLevel: 'R0-read',
    defaultApprovalPolicy: 'A0-auto',
    executionMode: 'sync',
    scopeLimit: 'current-object',
    dataBoundary: 'local-only',
    concurrency: 'never_parallel',
    maxResultSizeChars: 5000,
    requiresEnv: [],
    requiresBinaries: [],
    isAsync: false,
  };

  readonly inputSchema = TodoInputSchema;
  readonly jsonSchema = TODO_JSON_SCHEMA;

  async execute(
    input: TodoInput,
    _ctx: ToolExecutionContext,
  ): Promise<ToolExecutionResult<TodoOutput>> {
    const start = Date.now();

    switch (input.action) {
      case 'list': {
        const items = [...this.items.values()];
        const rawText = items.length === 0
          ? 'No tasks.'
          : items.map((t) =>
              `[${t.status}] ${t.id}: ${t.title} (${t.priority})`
            ).join('\n');
        return this.ok({ items, action: 'list' }, rawText, Date.now() - start);
      }

      case 'add': {
        if (!input.title) return this.fail('Title is required for add action', Date.now() - start);
        this.counter++;
        const id = input.id ?? `task-${this.counter}`;
        const item: TodoItem = {
          id,
          title: input.title,
          description: input.description,
          status: input.status ?? 'pending',
          priority: input.priority ?? 'medium',
          createdAt: new Date().toISOString(),
        };
        this.items.set(id, item);
        return this.ok(
          { items: [item], action: 'add', affected: id },
          `Added: ${id} - ${input.title}`,
          Date.now() - start,
        );
      }

      case 'update': {
        if (!input.id) return this.fail('ID is required for update action', Date.now() - start);
        const existing = this.items.get(input.id);
        if (!existing) return this.fail(`Task "${input.id}" not found`, Date.now() - start);
        const updated: TodoItem = {
          ...existing,
          title: input.title ?? existing.title,
          description: input.description ?? existing.description,
          status: input.status ?? existing.status,
          priority: input.priority ?? existing.priority,
        };
        this.items.set(input.id, updated);
        return this.ok(
          { items: [updated], action: 'update', affected: input.id },
          `Updated: ${input.id} → ${updated.status}`,
          Date.now() - start,
        );
      }

      case 'remove': {
        if (!input.id) return this.fail('ID is required for remove action', Date.now() - start);
        const removed = this.items.delete(input.id);
        if (!removed) return this.fail(`Task "${input.id}" not found`, Date.now() - start);
        return this.ok(
          { items: [], action: 'remove', affected: input.id },
          `Removed: ${input.id}`,
          Date.now() - start,
        );
      }

      case 'clear': {
        const count = this.items.size;
        this.items.clear();
        return this.ok(
          { items: [], action: 'clear' },
          `Cleared ${count} tasks.`,
          Date.now() - start,
        );
      }
    }
  }
}
```

### 3.1 Shell 命令安全策略

#### 3.1.1 破坏性命令检测

```typescript
/**
 * 破坏性命令模式。
 * 匹配到这些模式的命令将被升级到 R3-external-write 风险等级。

export const DESTRUCTIVE_PATTERNS: readonly { pattern: RegExp; id: string; description: string }[] = [
  // 文件删除

  { pattern: /\brm\s+-rf\b/i, id: 'rm-rf', description: 'Recursive forced deletion' },

  { pattern: /\bmv\s+.*\s+\/dev\/null/i, id: 'mv-devnull', description: 'Move to /dev/null' },

  // Git 危险操作
  { pattern: /\bgit\s+reset\s+--hard/i, id: 'git-reset-hard', description: 'Hard git reset' },

  { pattern: /\bchown\b/i, id: 'chown', description: 'Ownership change' },
  { pattern: /\bdd\s+/i, id: 'dd', description: 'Low-level disk operation' },
  { pattern: /\bmkfs\b/i, id: 'mkfs', description: 'Filesystem creation' },
  { pattern: /\bformat\b/i, id: 'format', description: 'Disk formatting' },

  // 网络出境
  { pattern: /\bcurl\s+.*-X\s*(POST|PUT|DELETE|PATCH)/i, id: 'curl-write', description: 'HTTP write request' },
  { pattern: /\bwget\b/i, id: 'wget', description: 'File download' },

  // 权限提升
  { pattern: /\bsudo\b/i, id: 'sudo', description: 'Privilege escalation' },
  { pattern: /\bsu\s+/i, id: 'su', description: 'User switch' },

  // 包管理（全局安装）
  { pattern: /\bnpm\s+install\s+-g\b/i, id: 'npm-global', description: 'Global npm install' },
  { pattern: /\bpip\s+install\b(?!.*--user)/i, id: 'pip-install', description: 'Pip install (may be global)' },

  // 进程管理
  { pattern: /\bkill\s+-9\b/i, id: 'kill-9', description: 'Force kill process' },
  { pattern: /\bkillall\b/i, id: 'killall', description: 'Kill all processes by name' },

  // 环境变量泄露
  { pattern: /\benv\b|\bprintenv\b/i, id: 'env-print', description: 'Environment variable exposure' },
  { pattern: /\bset\s*$/im, id: 'set-all', description: 'Print all shell variables' },
];

/**
 * 只读命令模式。匹配到这些模式的命令保持 R0-read。
 */
export const READONLY_PATTERNS: readonly RegExp[] = [
  /^\s*(ls|dir|pwd|echo|cat|head|tail|less|more|wc|file|stat|du|df)\b/i,
  /^\s*(find|locate|which|whereis|type)\b/i,
  /^\s*(grep|egrep|fgrep|rg|ag|ack)\b/i,
  /^\s*(date|cal|uptime|whoami|hostname|uname)\b/i,
  /^\s*git\s+(status|log|diff|show|branch|tag|remote)\b/i,
  /^\s*(node|python3?|ruby|go)\s+--version\b/i,
  /^\s*(npm|yarn|pnpm)\s+(list|ls|info|view|outdated)\b/i,
];

/**
 * 根据命令内容分类风险等级。
 */
export function classifyCommandRisk(command: string): RiskLevel {
  const trimmed = command.trim();

  // 1. 破坏性命令 → 最高风险
  for (const { pattern } of DESTRUCTIVE_PATTERNS) {
    if (pattern.test(trimmed)) {
      return 'R3-external-write';
    }
  }

  // 2. 只读命令 → 最低风险
  for (const pat of READONLY_PATTERNS) {
    if (pat.test(trimmed)) {
      return 'R0-read';
    }
  }

  // 3. 默认：内部写入
  return 'R1-internal-write';
}

/**
 * 检测命令中的破坏性模式，返回匹配到的威胁列表。
 */
export function detectDestructivePatterns(
  command: string,
): readonly { id: string; description: string }[] {
  const threats: { id: string; description: string }[] = [];
  for (const { pattern, id, description } of DESTRUCTIVE_PATTERNS) {
    if (pattern.test(command)) {
      threats.push({ id, description });
    }
  }
  return threats;
}
```

```typescript
import path from 'node:path';

/**
 * 验证工作目录是否在允许范围内。
 * 防止 terminal 工具在敏感目录中执行命令。
 */
export function validateWorkingDirectory(
  requestedCwd: string,
  allowedPaths: readonly string[],
): { valid: boolean; reason?: string } {
  const resolved = path.resolve(requestedCwd);

  const isAllowed = allowedPaths.some((allowed) => {
    const resolvedAllowed = path.resolve(allowed);
    return resolved === resolvedAllowed || resolved.startsWith(resolvedAllowed + path.sep);
  });

  if (!isAllowed) {
    return {
      valid: false,
      reason: `Directory "${resolved}" is outside allowed paths: [${allowedPaths.join(', ')}]`,
    };
  }

  return { valid: true };
}
```

### 3.2 文件操作的路径安全

```typescript
// packages/agent-core/src/tools/safety/path-validator.ts

import path from 'node:path';
import fs from 'node:fs';

/**
 * 敏感路径黑名单。
 * 这些路径下的文件永远不允许读写，即使在 allowedPaths 范围内。
 */
export const SENSITIVE_PATH_PATTERNS: readonly RegExp[] = [
  /[/\\]\.ssh[/\\]/i,
  /[/\\]\.gnupg[/\\]/i,
  /[/\\]\.env(\.[a-z]+)?$/i,
  /[/\\]\.env\.local$/i,
  /[/\\]\.aws[/\\]credentials/i,
  /[/\\]\.docker[/\\]config\.json/i,
  /[/\\]\.kube[/\\]config/i,
  /[/\\]\.netrc$/i,
  /[/\\]\.npmrc$/i,
  /[/\\]\.pypirc$/i,
  /[/\\]\.git-credentials$/i,
  /[/\\]keychain[/\\]/i,
  /[/\\]\.password-store[/\\]/i,
  /[/\\]\.mozilla[/\\]/i,
  /[/\\]\.chrome[/\\]/i,
  /[/\\]Library[/\\]Keychains[/\\]/i,
];

export interface PathValidationResult {
  readonly valid: boolean;
  readonly reason?: string;
}

/**
 * 验证文件路径的安全性。
 *
 * 检查项：
 * 1. 路径遍历防护（不允许 ../.. 逃逸 allowedPaths）
 * 2. 符号链接安全（解析后仍需在 allowedPaths 内）
 * 3. 敏感路径黑名单
 */
export function validatePath(
  targetPath: string,
  allowedPaths: readonly string[],
): PathValidationResult {
  const resolved = path.resolve(targetPath);

  // 1. 路径遍历检查
  const isWithinAllowed = allowedPaths.some((allowed) => {
    const resolvedAllowed = path.resolve(allowed);
    return resolved === resolvedAllowed || resolved.startsWith(resolvedAllowed + path.sep);
  });

  if (!isWithinAllowed) {
    return {
      valid: false,
      reason: `Path "${resolved}" is outside allowed directories`,
    };
  }

  // 2. 敏感路径检查
  for (const pattern of SENSITIVE_PATH_PATTERNS) {
    if (pattern.test(resolved)) {
      return {
        valid: false,
        reason: `Path "${resolved}" matches sensitive path pattern: ${pattern.source}`,
      };
    }
  }

  // 3. 符号链接安全检查
  try {
    const realPath = fs.realpathSync(resolved);
    if (realPath !== resolved) {
      const realIsAllowed = allowedPaths.some((allowed) => {
        const resolvedAllowed = path.resolve(allowed);
        return realPath === resolvedAllowed || realPath.startsWith(resolvedAllowed + path.sep);
      });

      if (!realIsAllowed) {
        return {
          valid: false,
          reason: `Symlink "${resolved}" resolves to "${realPath}" which is outside allowed directories`,
        };
      }

      for (const pattern of SENSITIVE_PATH_PATTERNS) {
        if (pattern.test(realPath)) {
          return {
            valid: false,
            reason: `Symlink "${resolved}" resolves to sensitive path "${realPath}"`,
          };
        }
      }
    }
  } catch {
    // 文件可能还不存在（写入场景），只检查父目录
  }

  return { valid: true };
}
```

```typescript
import { classifyCommandRisk, detectDestructivePatterns } from './command-classifier.js';
import { validatePath } from './path-validator.js';

/**
 * S2 工具安全检查层。
 * 作为 M3 责任链的一个 SafetyChecker 插入。
 * 在 PolicyChecker 之前运行（order = 1.5），针对内置工具的输入进行细粒度安全检查。

export class ToolSafetyChecker implements SafetyChecker {
  readonly layerId = 'tool-safety';
  readonly order = 1.5;

  async check(envelope: SafetyEnvelope): Promise<LayerCheckResult> {
    const toolName = envelope.capabilityCall.toolName;
    const args = envelope.capabilityCall.arguments;

    // Terminal 命令安全检查
    if (toolName === 'terminal' && typeof args['command'] === 'string') {
      const threats = detectDestructivePatterns(args['command'] as string);
      if (threats.length > 0) {
        envelope.threats.push(...threats.map((t) => `tool-safety:${t.id}`));

        if (risk === 'R3-external-write') {
          return {
            verdict: 'suspended',
            layerId: this.layerId,
            reason: `Destructive command detected: ${threats.map((t) => t.description).join(', ')}. Requires approval.`,
            threats: threats.map((t) => t.id),
          };
        }
      }
    }

    // 文件路径安全检查
    if (['file_read', 'file_write', 'file_patch'].includes(toolName)) {
      const filePath = args['path'] as string;
      if (filePath) {
        const result = validatePath(filePath, envelope.session.allowedPaths ?? [process.cwd()]);
        if (!result.valid) {
          return {
            verdict: 'denied',
            layerId: this.layerId,
            reason: result.reason ?? 'Path validation failed',

        }
```

## 第四章：工具并发执行

### 4.1 并发安全分类标准

每个工具通过 `concurrencySafety(input)` 声明其并发安全级别：

| 级别 | 语义 | 示例 |
|------|------|------|
| `parallel_safe` | 任意数量的实例可以并行运行 | web_search, web_extract, file_read, file_search, image_analyze |
| `path_scoped` | 操作不同路径时可并行，同一路径互斥 | file_write, file_patch, memory_tool |
| `never_parallel` | 全局互斥，同一时刻只能有一个实例运行 | terminal, clarify, todo, browser_interact |

### 4.2 并行执行器实现

```typescript
// packages/agent-core/src/tools/execution/parallel-executor.ts

import type { BuiltinTool, ToolExecutionContext, ToolExecutionResult } from '../types.js';

interface PendingToolCall {
  readonly tool: BuiltinTool;
  readonly input: unknown;
  readonly ctx: ToolExecutionContext;
  readonly callId: string;
}

interface ToolCallResult {
  readonly callId: string;
  readonly result: ToolExecutionResult;
}

/**
 * 并行工具执行器。
 * 根据工具的并发安全声明，智能调度并行执行。
 *
 * 算法：
 * 1. 将待执行的工具调用分为三组：parallel_safe, path_scoped, never_parallel
 * 2. parallel_safe 组全部并行执行（受 maxWorkers 限制）
 * 3. path_scoped 组按路径分桶，同桶串行，不同桶并行
 * 4. never_parallel 组严格串行执行
 * 5. 三组按顺序执行：parallel_safe → path_scoped → never_parallel
 */
export class ParallelToolExecutor {
  private readonly maxWorkers: number;

  constructor(maxWorkers = 8) {
    this.maxWorkers = maxWorkers;
  }

  async executeBatch(calls: readonly PendingToolCall[]): Promise<readonly ToolCallResult[]> {
    const parallelSafe: PendingToolCall[] = [];
    const pathScoped: PendingToolCall[] = [];
    const neverParallel: PendingToolCall[] = [];

    for (const call of calls) {
      const level = call.tool.concurrencySafety(call.input);
      switch (level) {
        case 'parallel_safe': parallelSafe.push(call); break;
        case 'path_scoped': pathScoped.push(call); break;
        case 'never_parallel': neverParallel.push(call); break;
      }
    }

    const results: ToolCallResult[] = [];

    // Phase 1: parallel_safe
    if (parallelSafe.length > 0) {
      results.push(...await this.executeParallel(parallelSafe));
    }

    // Phase 2: path_scoped
    if (pathScoped.length > 0) {
      results.push(...await this.executePathScoped(pathScoped));
    }

    // Phase 3: never_parallel
    for (const call of neverParallel) {
      results.push(await this.executeSingle(call));
    }

    return results;
  }

  private async executeParallel(calls: PendingToolCall[]): Promise<ToolCallResult[]> {
    const results: ToolCallResult[] = [];
    const chunks = this.chunk(calls, this.maxWorkers);

    for (const chunk of chunks) {
      const batchResults = await Promise.all(
        chunk.map((call) => this.executeSingle(call)),
      );
      results.push(...batchResults);
    }

    return results;
  }

  private async executePathScoped(calls: PendingToolCall[]): Promise<ToolCallResult[]> {
    // 按路径分桶
    const buckets = new Map<string, PendingToolCall[]>();

    for (const call of calls) {
      const paths = call.tool.scopePaths(call.input);
      const key = paths.sort().join('|') || '__default__';
      const bucket = buckets.get(key) ?? [];
      bucket.push(call);
      buckets.set(key, bucket);
    }

    // 检查路径冲突
    const conflictGroups = this.detectPathConflicts(buckets);
    const results: ToolCallResult[] = [];

    const parallelBuckets: PendingToolCall[][] = [];
    const serialBuckets: PendingToolCall[][] = [];

    for (const [key, bucket] of buckets) {
      if (conflictGroups.has(key)) {
        serialBuckets.push(bucket);
      } else {
        parallelBuckets.push(bucket);
      }
    }

    // 并行执行无冲突桶
    if (parallelBuckets.length > 0) {
      const promises = parallelBuckets.map(async (bucket) => {
        const bucketResults: ToolCallResult[] = [];
        for (const call of bucket) {
          bucketResults.push(await this.executeSingle(call));
        }
        return bucketResults;
      });
      const batchResults = await Promise.all(promises);
      for (const batch of batchResults) results.push(...batch);
    }

    // 串行执行冲突桶
    for (const bucket of serialBuckets) {
      for (const call of bucket) {
        results.push(await this.executeSingle(call));
      }
    }

    return results;
  }

  /**
   * 路径冲突检测算法。
   * 两个桶冲突 = 它们的路径集合存在交集或前缀包含关系。
   */
  private detectPathConflicts(buckets: Map<string, PendingToolCall[]>): Set<string> {
    const conflicts = new Set<string>();
    const keys = [...buckets.keys()];

    for (let i = 0; i < keys.length; i++) {
      for (let j = i + 1; j < keys.length; j++) {
        const pathsA = keys[i].split('|');
        const pathsB = keys[j].split('|');

        if (this.pathsConflict(pathsA, pathsB)) {
          conflicts.add(keys[i]);
          conflicts.add(keys[j]);
        }
      }
    }

    return conflicts;
  }

  private pathsConflict(pathsA: string[], pathsB: string[]): boolean {
    for (const a of pathsA) {
      for (const b of pathsB) {
        if (a === b || a.startsWith(b + '/') || b.startsWith(a + '/')) {
          return true;
        }
      }
    }
    return false;
  }

  private async executeSingle(call: PendingToolCall): Promise<ToolCallResult> {
    try {
      const result = await call.tool.execute(call.input, call.ctx);
      return { callId: call.callId, result };
    } catch (err: unknown) {
      return {
        callId: call.callId,
        result: {
          success: false,
          output: undefined as unknown,
          rawText: '',
          truncated: false,
          durationMs: 0,
          error: err instanceof Error ? err.message : String(err),
        },
      };
    }
  }

  private chunk<T>(arr: T[], size: number): T[][] {
    const chunks: T[][] = [];
    for (let i = 0; i < arr.length; i += size) {
      chunks.push(arr.slice(i, i + size));
    }
    return chunks;
  }
}
```

```typescript
  private disabledTools: string[] = [];

  setTimeout(ms: number): this { this.globalTimeoutMs = ms; return this; }

    return {
      name: this.name,
```

```typescript
    concurrencySafety() { return metadata.concurrency; },
    scopePaths() { return []; },

      input: unknown,
      _ctx: ToolExecutionContext,

          truncated: false, durationMs: Date.now() - start,
          error: err instanceof Error ? err.message : String(err),
        };
      }
    },
  };
```

```typescript
// packages/agent-core/src/tools/result/truncation.ts

export interface TruncationStrategy {
  readonly name: string;
  truncate(text: string, maxChars: number): TruncatedResult;
}

export interface TruncatedResult {
  readonly text: string;
  readonly truncated: boolean;
  readonly originalLength: number;
  readonly strategy: string;
}

/** 简单截断：直接截取前 N 个字符 */
export class SimpleTruncation implements TruncationStrategy {
  readonly name = 'simple';

  truncate(text: string, maxChars: number): TruncatedResult {
    if (text.length <= maxChars) {
      return { text, truncated: false, originalLength: text.length, strategy: this.name };
    }
    return {
      text: text.slice(0, maxChars) + `\n\n... [truncated: ${maxChars}/${text.length} chars]`,
      truncated: true,
      originalLength: text.length,
      strategy: this.name,
    };
  }
}

/** 智能截断：保留首尾内容，压缩中间部分 */
export class SmartTruncation implements TruncationStrategy {
  readonly name = 'smart';
  private readonly headRatio: number;
  private readonly tailRatio: number;

  constructor(headRatio = 0.6, tailRatio = 0.3) {
    this.headRatio = headRatio;
    this.tailRatio = tailRatio;
  }

  truncate(text: string, maxChars: number): TruncatedResult {
    if (text.length <= maxChars) {
      return { text, truncated: false, originalLength: text.length, strategy: this.name };
    }

    const headSize = Math.floor(maxChars * this.headRatio);
    const tailSize = Math.floor(maxChars * this.tailRatio);
    const omitted = text.length - headSize - tailSize;

    return {
      text: `${text.slice(0, headSize)}\n\n... [${omitted} chars omitted] ...\n\n${text.slice(-tailSize)}`,
      truncated: true,
      originalLength: text.length,
      strategy: this.name,
    };
  }
}

/** 行级截断：保留前 N 行和后 M 行 */
export class LineTruncation implements TruncationStrategy {
  readonly name = 'line';

  truncate(text: string, maxChars: number): TruncatedResult {
    if (text.length <= maxChars) {
      return { text, truncated: false, originalLength: text.length, strategy: this.name };
    }

    const lines = text.split('\n');
    const maxLines = Math.max(10, Math.floor(maxChars / 80));
    const headLines = Math.floor(maxLines * 0.7);
    const tailLines = Math.floor(maxLines * 0.3);

    if (lines.length <= maxLines) {
      return new SmartTruncation().truncate(text, maxChars);
    }

    const head = lines.slice(0, headLines).join('\n');
    const tail = lines.slice(-tailLines).join('\n');
    const omittedLines = lines.length - headLines - tailLines;

    return {
      text: `${head}\n\n... [${omittedLines} lines omitted] ...\n\n${tail}`,
      truncated: true,
      originalLength: text.length,
      strategy: this.name,
    };
  }
}

/** 根据工具类型选择最佳截断策略 */
export function selectTruncationStrategy(toolName: string): TruncationStrategy {
  switch (toolName) {
    case 'terminal':
    case 'code_execute':
      return new LineTruncation();
    case 'web_extract':
    case 'browser_snapshot':
      return new SmartTruncation();
    default:
      return new SimpleTruncation();
  }
}
```

```typescript
// packages/agent-core/src/tools/result/result-storage.ts

import fs from 'node:fs/promises';
import path from 'node:path';
import crypto from 'node:crypto';

/**
 * 工具结果存储。
 * 当结果超过截断限制时，将完整结果持久化到磁盘，
 * 在返回给 LLM 的截断结果中附加引用 ID。
 * 参考 Hermes 的 tool_result_storage 设计。
 */
export class ToolResultStorage {
  private readonly storageDir: string;

  constructor(storageDir: string) {
    this.storageDir = storageDir;
  }

  async store(
    toolName: string,
    sessionId: string,
    result: string,
  ): Promise<{ refId: string; storagePath: string }> {
    const refId = `tr_${crypto.randomBytes(8).toString('hex')}`;
    const dir = path.join(this.storageDir, sessionId);
    await fs.mkdir(dir, { recursive: true });

    const storagePath = path.join(dir, `${refId}.txt`);
    await fs.writeFile(storagePath, result, 'utf-8');

    const metaPath = path.join(dir, `${refId}.meta.json`);
    await fs.writeFile(
      metaPath,
      JSON.stringify({
        refId, toolName, sessionId,
        storedAt: new Date().toISOString(),
        sizeBytes: Buffer.byteLength(result),
      }),
      'utf-8',
    );

    return { refId, storagePath };
  }

  async retrieve(sessionId: string, refId: string): Promise<string | null> {
    const storagePath = path.join(this.storageDir, sessionId, `${refId}.txt`);
    try {
      return await fs.readFile(storagePath, 'utf-8');
    } catch {
      return null;
    }
  }

  async cleanup(sessionId: string): Promise<void> {
    const dir = path.join(this.storageDir, sessionId);
    try {
      await fs.rm(dir, { recursive: true, force: true });
    } catch {
      // 目录不存在，忽略
    }
  }
}
```

```typescript
 * 将 ToolExecutionResult 映射为 M1 事件流中的 CapabilityEvent。
 */
export function toolResultToEvents<Output>(

  result: ToolExecutionResult<Output>,
  refId?: string,

  const capabilityId = `builtin.*.${toolName}`;

  if (result.success) {
    events.push({
      kind: 'completed',
      capabilityId,
      timestamp: new Date().toISOString(),
      partialResult: result.output as Partial<Output>,

    });
  } else {
    events.push({
      kind: 'failed',
      capabilityId,
      timestamp: new Date().toISOString(),
      error: result.error ?? 'Unknown error',
    });
  }

  return events;
}
```

```typescript
/**
 * 将底层错误转换为用户友好的错误消息。

export function formatToolError(
  toolName: string,

      userMessage: `Network error: Could not connect to the server.`,
      suggestion: 'Check your internet connection and try again.',
    };

    return {
      userMessage: `The operation timed out.`,
      suggestion: 'Try again with a longer timeout or simplify the request.',
    };
  }

  if (error.includes('EACCES') || error.includes('EPERM')) {
    return {
      userMessage: `Permission denied.`,
      suggestion: 'Check file permissions or try a different path.',
    };

  if (error.includes('ENOENT')) {

      userMessage: `File or directory not found.`,
      suggestion: 'Verify the path is correct. Use file_search to find the file.',
    };
  }

  if (error.includes('API_KEY') || error.includes('api_key') || error.includes('unauthorized')) {
    return {
      userMessage: `Authentication required for ${toolName}.`,
      suggestion: 'Set the required API key in your environment variables.',
    };
  }

  return {
    userMessage: `${toolName} failed: ${error.slice(0, 200)}`,
```
