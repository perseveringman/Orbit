# 18-设计系统与换肤

## 1. 概述
Orbit 的设计系统完全参照 Capacities 的设计规范进行构建。采用基于 **OKLCH** 色彩空间的语义化 Token 系统，支持明暗双色主题无缝切换，并提供高度一致的排版、间距、阴影和圆角规范。整体布局采用经典的三栏式桌面端/Web端布局。

## 2. 布局系统 (Layout System)

应用采用全屏三栏弹性布局（Three-Panel Layout）：

```text
┌──────────────┬──────────────────────────────┬─────────────────┐
│   Sidebar    │       Main Content           │  Right Panel    │
│   288px      │   flex: 1 (adaptive)         │  431px (固定)    │
│   fixed      │   min-width: 0               │  flex-shrink: 0 │
└──────────────┴──────────────────────────────┴─────────────────┘
```

### 2.1 左侧边栏 (Sidebar)
- **宽度**: 固定 `288px`
- **背景**: `var(--bg-back)`
- **边框**: 右侧 `0.5px solid var(--border-base)`
- **结构**: 
  - 顶部：Logo/标题区 + "New Object" 全局新建按钮
  - 导航区：日历 (Calendar)、入门指南 (Getting started)、固定项 (Pinned)、对象类型 (Object types)
  - 底部：工具栏（设置、主题切换、用户栏）与辅助链接（回收站、文档等）

### 2.2 主内容区 (Main Content)
- **宽度**: 自适应 (`flex: 1`)，设置 `min-width: 0` 防止内容溢出
- **背景**: 透明（透出 `var(--bg-back)` 或底层背景）
- **结构**:
  - **顶部导航 (Page Header)**: 最小高度 `58px`，包含标题、前后导航、视图切换器 (View Switcher) 以及右侧操作栏（搜索、排序、新建）。
  - **标签栏 (Tab Bar)**: 高度 `40px`，用于视图内的子分类过滤（如 Overview / All）。
  - **内容主体 (Page Body)**: `padding: 24px`，内部可滚动。

### 2.3 右侧面板 (Right Panel)
- **宽度**: 固定 `431px`
- **背景**: `var(--bg-back)`
- **边框**: 左侧 `0.5px solid var(--border-base)`
- **结构**:
  - 顶部：紧凑型标签栏（如 Graph view, Daily Notes 等）
  - 内容区：图谱可视化、关联笔记列表等辅助信息。

## 3. 设计令牌 (Design Tokens)

### 3.1 色彩系统 (Color System)
采用 OKLCH 色彩空间，定义了明暗两套主题的语义化 Token。

#### 核心背景色
| Token | Light | Dark | 用途 |
|-------|-------|------|------|
| `--bg-back` | `oklch(0.986 0.002 67)` | `#1a1a1e` | 应用最底层背景 |
| `--bg-base` | `oklch(1.000 0.000 263)` | `#242428` | 内容基础背景 |
| `--bg-front` | `oklch(1.000 0.000 263)` | `#2a2a2e` | 卡片、浮层面板背景 |
| `--bg-el` | `oklch(0.968 0.002 67)` | `#3c3c42` | 元素背景（如激活的 Tab、Hover 状态） |
| `--bg-button-primary` | `oklch(0.389 0.005 301)` | `#e5e5e3` | 主按钮背景 |

#### 文本与边框
| Token | Light | Dark | 用途 |
|-------|-------|------|------|
| `--text-primary` | `oklch(0.219 0.006 286)` | `#ffffff` | 标题、主要文本 |
| `--text-secondary` | `oklch(0.389 0.005 301)` | `#a8a8a2` | 正文、导航文本 |
| `--text-subtle` | `oklch(0.573 0.005 34)` | `#6e6e68` | 占位符、计数、次要信息 |
| `--border-base` | `oklch(0.916 0.002 67)` | `#3c3c42` | 默认分割线、边框 |
| `--border-el-hover` | `oklch(0.716 0.006 31)` | `#6e6e68` | 悬停状态边框 |

#### 对象类型专属色 (恒定不随主题变化)
用于区分不同类型的数据（Project, Quote, Atomic note, Daily Note, Page, Tag），包含独立的背景、文本和边框颜色。例如：
- **Project**: Mint 绿色系
- **Quote**: Blush 红色系
- **Atomic note**: Cream 黄色系
- **Daily Note**: Periwinkle 蓝紫色系

### 3.2 排版 (Typography)
- **字体族**: 
  - Sans: `Inter, ui-sans-serif, system-ui`
  - Mono: `"Courier New", monospace`
- **字号阶层**: 基准字号 `15px` (`--text-base`)，覆盖从 `11px` (微小标签) 到 `48px` (Hero 标题) 的完整阶层。
- **字重**: `400` (常规), `500` (中等/激活), `600` (半粗), `700` (粗体标题)。

### 3.3 间距与圆角 (Spacing & Radius)
- **间距**: 基于 `4px` (`0.25rem`) 的系统。
- **圆角**: 
  - `--r-base` (8px): 最常用，用于按钮、导航项、下拉菜单。
  - `--r-xl` (12px) / `--r-2xl` (16px): 用于卡片、模态框、大面板。

### 3.4 阴影与动效 (Shadows & Motion)
- **阴影**: 采用 3 层叠加技术 (3-layer technique)，极低透明度 (0.01-0.05) 以实现自然的深度感。
  - `--shadow`: 默认对象卡片阴影。
  - `--shadow-2xl`: 模态框阴影。
- **过渡动效**: 默认时长 `0.15s`，缓动曲线 `cubic-bezier(.4,0,.2,1)`。

## 4. 核心组件 (Components)

1. **按钮 (Buttons)**
   - 主按钮 (Primary): 实色背景，圆角 8px。
   - 工具栏按钮 (Toolbar): 32x32px 幽灵按钮，Hover 时显现实色背景。
2. **对象卡片 (Object Cards)**
   - 包含类型标签 (Type Label)、大标题、预览文本和底部元数据 (Tags)。
   - Hover 时边框加深并提升阴影层级 (`--shadow-md`)。
3. **下拉菜单与视图切换 (View Switcher)**
   - 触发器带边框和背景，下拉面板带 `--shadow-lg` 阴影和 12px 圆角。
4. **模态框 (Modal / Command Palette)**
   - 居中偏上布局，带深色模糊背景 (`backdrop-filter: blur(4px)`)。
   - 内部包含搜索输入框和带图标的列表项。
5. **详情编辑器 (Object Detail)**
   - 全屏覆盖层 (Overlay)，最大宽度 720px 居中阅读区。
   - 超大标题 (36px)，无缝融合属性编辑与正文编辑。

## 5. 换肤机制 (Theming Implementation)

1. **CSS 变量驱动**: 所有颜色均通过 CSS Custom Properties 定义在 `:root` 下。
2. **主题切换**: 
   - 通过在 `<html>` 标签上切换 `data-theme="light"` 或 `data-theme="dark"` 属性来实现。
   - 配合 Tailwind CSS v4 的机制，或者直接在全局 CSS 中编写 `[data-theme="dark"] { ... }` 覆盖变量。
3. **持久化**: 用户的主题偏好通过 `localStorage` 保存，并在应用初始化时同步读取应用。
